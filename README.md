# Unsplash
Unsplash Image API를 활용한 사진 앱

## 커밋 메시지
- Convention
  - [Feat] : 기능 추가 / 새로운 로직
  - [Fix] : 버그 수정
  - [Chore] : 간단한 수정
  - [Docs] : 문서 및 리드미 작성
  - [Refactor] : 리팩토링
